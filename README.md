# Sistema de Controle de Multas

## Descrição
Sistema desenvolvido em Python para cadastrar, gerenciar e gerar relatórios de multas de veículos.

## Tecnologias
- Python 3.x

## Estrutura do Projeto
- cadastro.py → módulo de cadastro de multas
- gerenciamento.py → módulo de gerenciamento
- relatorio.py → módulo de relatórios
- main.py → arquivo principal
